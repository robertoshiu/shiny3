// 04 · CULTURE & VALUES

拿授權，拿責任

我們不是一家用流程規避責任的公司。每一個角色都有清晰的 RACI A（Accountable）— 你對自己的領域的決策結果負責，也擁有推動它所需的全部授權。

04
// PRINCIPLE 01

授權不是口號

每個技術決策都有且只有一個 A（Accountable Owner）。沒有「上面還要核准」的迴圈——如果你是 A，你就有最終拍板權，也承擔結果。我們以合約 Gate 確認，不以層級審批代替。

// PRINCIPLE 02

責任由架構承諾

SLO ≥ 95%、RTO ≤ 4hr、良率 ≥ 90% 不是 KPI 口號，而是寫入合約的承諾。這代表每一個系統設計都有後果，我們的工程師不只交付，更守住業務連續性。

// PRINCIPLE 03

數字說話，風險透明

三級風險升級（紅/黃/綠），DR 季度演練納入 Review 閉環，跨廠模板 100% 歸檔。沒有隱藏問題的文化，只有面對問題的工具與流程。
